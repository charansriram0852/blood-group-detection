# 🩸 Blood Group Detection Using Fingerprint

## 📌 Overview

This project aims to predict a person's **blood group** using fingerprint images with the help of **Deep Learning techniques**. It leverages image processing and a trained neural network model to classify blood groups based on fingerprint patterns.

---

## 🚀 Features

* 🔍 Fingerprint image analysis
* 🤖 Deep Learning model (ResNet-based)
* 📊 Blood group classification
* 💡 Easy to use and extend

---

## 🛠️ Technologies Used

* Python 🐍
* TensorFlow / Keras
* OpenCV
* NumPy
* Matplotlib

---

## 📂 Project Structure

* `model_blood_group_detection_resnet.keras` → Trained model
* `*.py` files → Code implementation
* `dataset/` → Fingerprint images (if included)

---

## ⚙️ How It Works

1. Input fingerprint image
2. Image preprocessing (resizing, normalization)
3. Model prediction using trained ResNet
4. Output predicted blood group

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/charansriram0852/blood-group-detection.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the main file:

```bash
python main.py
```

---

## 📸 Output Example

* Input: Fingerprint image
* Output: Predicted blood group (A, B, AB, O)
